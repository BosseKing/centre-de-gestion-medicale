Ce dossier correspond à un service spécifique du projet SOA "Centre de gestion médicale".

## Nom du dossier
Le nom du dossier doit refléter clairement le service qu'il contient, par exemple :
- `auth-service` : service d'authentification
- `patient-service` : gestion des patients
- `consultation-service` : gestion des consultations
- `billing-service` : gestion de la facturation
